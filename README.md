### 💻 Comandos básicos do Git
Aprendi a utilizar os principais comandos utilizados no dia a dia de um desenvolvedor:

- `git init` → criar um repositório Git.
- `git clone` → clonar um repositório existente.
- `git add` → adicionar arquivos para controle de versão.
- `git commit` → registrar alterações no histórico.
- `git status` → verificar o estado dos arquivos.
- `git log` → visualizar histórico de commits.

---

### 🌐 Uso do GitHub
Também aprendi a utilizar o GitHub como plataforma de hospedagem de código:

- Criar e gerenciar **repositórios remotos**.

- Trabalhar com **forks** e colaboração em projetos.
